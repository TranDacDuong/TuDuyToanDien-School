Cách lấy token test

Token cần quyền gì?

Cần Page Access Token của Fanpage, có quyền pages_manage_posts và thường cần thêm pages_show_list.

Các bước test nhanh
  1. Mở Meta Graph API Explorer.
  2. Chọn app Meta của MindUp hoặc tạo app test.
  3. Lấy User Token có quyền quản lý Page.
  4. Đổi sang Page Access Token của “MindUp - Tư Duy Sinh Học”.
  5. Dán token vào trang này và đăng thử.
